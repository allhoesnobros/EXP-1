Theory :-

In python we can declare multiple data type variables.

for eg. a = 2, b = 4.324 , c = "APPLE" , d = True

We can display data type of the variable by using type() command.

for eg. print(type(a))

In python there are 2 ways to display multiple data in output. By using multiple print command or by declearing variable (such as a,b,c) and using print(a,b,c). We can use print(a,b,c,sep="\n") command to display data stored in variable on new line in output.

Basic mathematical operator are built-in in python such as addition(+),subtraction(-),multiplication(*),division(/),power(**),etc.

To find sum of two integer a and b we can use a+b and to display it in ouput use print("SUM =",a+b). The same can be used for other mathematical operations with there desingnated symbols.

In the above the mathematical operations the value of the integers were pre decleared and if you want to find sum of other integers either you have to copy and paste code and change the integer values or change the value in the current code. Instead of this we can take value from the user and print the required output.

To take input from the user we have to declare the data type of the input and we can also the display what we want from the user (such as "Enter the value of a").
